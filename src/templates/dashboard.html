How to export your chat history
  1. Open Telegram Desktop
  2. Navigate to the group chat
  3. Click the ⋮ menu (top-right)
  4. Select Export chat history
  5. Set format to Machine-readable JSON
  6. Choose a date range and click Export
Requires Telegram Desktop — not available on mobile.
